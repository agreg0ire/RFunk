rmFiles

mixed rmFiles([string root_dir ])

Description

Supprime récursivement tous les fichiers à partir d'un répertoire racine.
Si une erreur de suppresion survient pour diverses raisons : fichier utilisé en ressource, fichier caché ou fichier système etc,
sera renvoyé un tableau contenant le chemin vers le(s) fichier(s) en question, sinon VRAI sera retourné.

Exemple



$m_retour=$o_rfunk->rmFiles('c:\windows');

if(is_array($m_retour))
{
	echo 'Ces fichiers n ont pas pu être supprimés => ';

	foreach($m_retour as $i_key => $s_value): 
	
		echo '<br />'.$s_value; 
		
	endforeach;

}elseif(is_bool($m_retour))
{

	echo 'Tous les fichiers ont bien été supprimés !';
}
