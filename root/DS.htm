RFunk::DS

string RFunk::DS

Description

Renvoi la constante DIRECTORY_SEPARATOR (qui on le rappel varie entre windows et unix).
C'est une constante de classe, il n'est donc pas utile de la redéfinir en dehors de la classe.
(Utilisez l'opérateur de résolution de porté pour y accéder.)

Exemple



echo RFunk::DS; //renvoi "\" si vous êtes sous windows ou "/" sous Unix
