rmParentDir

NULL rmParentDir( string root_dir )

Description

Supprime le dossier parent récursivement à partir d'un répertoire racine si ce dernier est bien sûr vide !

Utilisez rmFiles() pour vider tous les dossiers, puis getDirsPaths() pour récupérer l'arboresence dans un tableau.
Encore une fois, un chemin absolu est nécessaire car une recherche de charactère est opérée sur la constante
DIRECTORY_SEPARATOR (qui d'ailleurs varie entre Unix et Windows).

Renvoi le chemin vers le premier dossier parent qui n'est pas vide.

Exemple


$o_rfunk=new RFunk;

$m_retour=$o_rfunk->rmFiles('C:\Documents and Settings\pouet\Mes documents\Téléchargements');

if(is_array($m_retour))
{
	echo 'Ces fichiers n ont pas pu être supprimés => <br />';

	foreach($m_retour as $i_key => $s_value): 
	
		echo <br />'.$s_value; 
		
	endforeach;

}elseif(is_bool($m_retour))
{

	$a_dir_paths=$o_rfunk->getDirsPaths('C:\Documents and Settings\pouet\Mes documents\Téléchargements');


	foreach($a_dir_paths as $i_key => $s_value)
	{
		$o_rfunk->rmParentDir($s_value);
	}

}
