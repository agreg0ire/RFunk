zipFiles

mixed zipFiles( string root_dir [, string root_dest [, string zip_names [, integer max_file_size ]]])

Description

A chaque appel récursif, c'est à dire à chaque dossier, cette méthode va archiver tous les fichiers y étant présents
avec un nom correspondant à l'incrémentation des appels récursifs.
Par défaut, le répertoire de déstination vaut le dossier courant.
La méthode renvoi un tableau avec les fichiers supérieurs à votre limite indiquée s'il y en eu, sinon VRAI.

Exemple


$m_retour=$o_rfunk->zipFiles('c:\Documents and settings\user\bureau',getcwd(),'retour_archivage', 6);

if(is_bool($m_retour))
{
	echo 'Tout vos fichiers ont été archivés';
	
}elseif(is_array($m_retour))
{
		echo 'Voici le(s) fichiers qui n ont pas été archivés =>';

		foreach($m_retour as $i_key => $s_value)
		{
				echo '<br />'.$s_value;
		}

}
