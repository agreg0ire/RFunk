unZipFolders

mixed unZipFolders( string root_dir [, string root_dest [, integer max_size_zip [, integer max_size_file_in_zip [, boolean extract_in_same_dir [, boolean extract_only_file ]]]]])

Description

Cette méthode qui dispose de nombreux paramètres optionnels permet d'extraire chaque archive ZIP trouvée dans une aroborescence à partir d'un répertoire root
Le premier paramètre est obligatoire, par défaut la destination d'extraction se fait dans le répertoire d'exécution du script,
Seront traités que les ZIPS inférieurs ou égal à 5mo et dedans les fichiers inférieurs ou égal à 1 mo.

Vous pouvez augmentez ces valeurs à vos risques et périls ! (PHP n'est pas vraiment fait pour ce genre de traitement).

Enfin, les deux derniers booléens optionnels correspondent successivement et respectivement à :
- extraction dans un dossier unique (par défaut), ou dans différents dossiers correspondant au compteur de récursivité.(false)
- extraction des fichiers uniquement (par défaut), ou conserver l'arborescence interne au zip avec, dossiers. (false)

La méthode renvoi un tableau avec le "dirname" des zips s'il y en a eu dans l'arborescence spécifiée par votre recherche ou faux si aucun zip n'a été trouvé.

Astuce !
Dans votre dossier de destination, vous pouvez spécifier un ou plusieurs sous-dossiers non existant car la méthode "extractTo()" de la classe native PHP5 ZipArchive,
effectue automatiquement une création de dossier récursive correspondant à vos dossiers accueillant vos données extraites, si ces dossiers n'existent pas.

Exemple


$o_rfunk= new RFunk;

$m_retour=$o_rfunk->unZipFolders('c:',getcwd().RFunk::DS.'dossier1_non_existant'.RFunk::DS.'dossier2_non_existant', 10, 2);

if(is_bool($m_retour))
{
	echo 'il n\'y avait pas de zip a partir de "c:"';
	
}elseif(is_array($m_retour))
{

		foreach($m_retour as $i_key => $s_value)
		{
				echo '<br />'.$s_value;
		}

}
