getAllExts

mixed getAllExts( [ string root_dir ])

Description

Renvoi toutes les extensions de fichiers existantes d'une arborescence à partir d'un répertoire racine.
Les extensions sont renvoyées au format type "mp3", "txt" etc. dans un tableau.
Par défaut la recherche s'effectue dans le répertoire courant.
S'il n'y avait que des dossiers la fonction renverra FAUX.

Exemple



$m_retour=$o_rfunk->getAllExts();

if(is_array($m_retour))
{
	foreach($m_retour as $i_key => $s_value): echo '<br />'.$s_value; endforeach;

}elseif(is_bool($m_retour))
{

	echo 'Votre répertoire racine ne contenait que des dossiers !';
}
