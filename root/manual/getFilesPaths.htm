getFilesPaths

mixed getFilesPaths( [, string root_dir ])

Description

Retourne un tableau contenant l'équivalent d'un dirname + basename pour chaque fichiers présents dans l'arborescence à partir d'un répertoire racine, ou FAUX si le dossier racine était vide.
Sans paramètre, la méthode démarrera dans le fichier courant.

Exemple



$m_retour=$o_rfunk->getFilesPaths('c:\wamp\www\test');

if(is_array($m_retour))
{
	foreach($m_retour as $i_key => $s_value): echo '<br />'.$s_value; endforeach;

}elseif(is_bool($m_retour))
{

	echo 'Votre répertoire racine était vide !';
}
