calculSize

integer calculSize( [ string root_dir [, bool arrondi au supérieur [, string seulement un type d'extension ]]])

Description

Renvoi le total du poids des fichiers à partir d'un répertoire racine. Par défaut, le calcul se fait dans le dossier courant et un arrondi se fait au MO près.
Attention, des résultats étranges peuvent survenir si des fichiers supérieurs à 3OO mo sont présents dans votre arborescence !
Par ailleurs, vous pouvez spécifier de ne calculez le poids total que pour un type de fichier.

Exemple



$i_retour=$o_rfunk->calculSize('c:\wamp\www\test\', false, 'rar');

var_dump($i_retour);
