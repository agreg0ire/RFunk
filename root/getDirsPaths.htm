getDirsPaths

mixed getDirsPaths( [, string root_dir ])

Description

Retourne un tableau contenant le chemin pointant vers chaque répertoires dans l'arborescence à partir d'un répertoire racine,
ou FAUX s'il n'y avait aucun sous-dossiers.
Sans paramètre, la méthode démarrera dans le fichier courant.

Attention au cas où vous voulez exploitez le résultat de cette méthode dans rmParentDir();,
VOUS DEVEZ IMPERATIVEMENT FOURNIR UN CHEMIN ABSOLU comme paramètre, répertoire racine !

Exemple

$o_rfunk->getDirsPaths(getcwd());
est préférable à 
$o_rfunk->getDirsPaths('.');
ou encore à 
$o_rfunk->getDirsPaths();
