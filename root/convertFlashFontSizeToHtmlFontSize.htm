convertFlashFontSizeToHtmlFontSize

mixed convertFlashFontSizeToHtmlFontSize( string text to format [, integer start compteur font-size from flash [, integer start compteur font-size html [, integer end compteur font-size flash ]]])

Description

Cette méthode est pour l'instant la dernière réalisée en date.
Elle n'est pas longue mais exploite bien certaines possibilités du langage.

Un double compteur incrémenté et "breaké" par une expression régulière. Cette méthode a été conçue dans le contexte d'un editeur de texte en AS3 avec update MySQL par flash remoting.
Comme les valeurs de font size générées par l'environnement auteur de flash ne correspondent pas avec celles d'affichage Html, Il fallait envisager une conversion élégante autrement que par un Switch Case qui se serait vite révélé fastidieux si la plage des valeurs en venait à augmenter.

La méthode renvoie le texte formaté avec les conversions sur le css font size effectuées, ou faux si aucune balise du type <FONT SIZE="X"> n'a été trouvées.
Par défaut, les valeurs de compteurs, sont à 8 pour le flash en initiliasation et 1 pour l'html, et 14 pour la fin du compteur flash, 7 pour le html .

Exemple



include('connexion_sql.php');

$o_rfunk=new RFunk;
	
$q_sel_info_by_rub=mysql_query('SELECT txt FROM flash_remoting WHERE rubrique="News"') or die(mysql_error());
	
	if(mysql_num_rows($q_sel_info_by_rub) > 0)
	{
		$a_fields_bdd=mysql_fetch_assoc($q_sel_info_by_rub);
		
	}
	
$s_retour_converted_fontsize_for_html_display =	$o_rfunk->convertFlashFontSizeToHtmlFontSize($a_fields_bdd['txt'],8, 1, 15);
