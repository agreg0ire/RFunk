constructeur

php_ini __construct([, integer max_execution_time])

Description

Défini un temps maximum d'exécution du script (par défaut sans limite) qui peut etre limité en fonction du besoin.
Si un type autre qu'un entier est passé en paramètre, le script s'arrètera automatiquement et affichera une erreur.

Exemple



$o_rfunk=new RFunk(60);
