chmodR

NULL chmodR( string root_dir , int chmod)

Description

Change l'attribut de tous les fichiers et dossiers récursivement.
La fonction chmod n'est pas activée chez tous les hébergeurs donc cette méthode est très limitée.

Exemple



$o_rfunk = new RFunk(0);

$o_rfunk->chmodR(getcwd(),0777);
